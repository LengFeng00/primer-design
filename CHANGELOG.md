# 变更日志

## [v4.0.0] - 2026-03-24

### 新增功能 ✨
- 面向对象重构：使用类封装，消除全局变量
- BLAST 批处理优化：支持批量查询，减少 I/O 开销
- 进度条显示：使用 tqdm 实时显示处理进度
- 完整日志系统：支持分级日志和文件记录
- 断点续传：支持程序中断后继续执行
- 配置文件支持：YAML 格式配置文件
- 错误重试机制：自动重试失败的操作
- 类型注解：完整的类型提示
- 单元测试：pytest 测试框架
- 依赖管理：requirements.txt

### 性能优化 🚀
- BLAST 批处理模式，减少临时文件创建
- 支持批量引物检查
- 优化多进程通信
- 检查点机制减少重复计算

### 改进 🎨
- 代码结构更清晰，易于维护
- 详细的错误信息和日志
- 更好的输入验证
- 支持多种输出格式（TSV/CSV/Excel）

### Bug 修复 🐛
- 修复多进程环境下 Fasta 对象序列化问题
- 修复 BLAST 临时文件清理不彻底的问题
- 改进异常处理，避免掩盖真实错误

### 文档 📚
- 完整的使用指南 (README_V4.md)
- 配置文件说明 (config.yaml)
- 单元测试示例 (tests/test_primer_design.py)
